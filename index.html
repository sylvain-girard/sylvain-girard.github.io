Chromaesthetor

creative code

Chromaesthetor is a web-based sound visualiser that allows you see your music. Everything you see is the result of analysing various components of sound data that is then used to drive visual elements and colour information.




Using the p5.Js library of Javascript, with p5.Sound for sound analysis, I created a number of iterations before combining some of them for the final result. The process essentially involved learning what information could be extracted from the sound data, using that to drive some basic visuals, then creating the final visuals driven by more simple data like the mouse position, before replacing that simple data with the sound data.
